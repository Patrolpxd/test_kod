About the Files

Material Wallpaper is a mobile wallpaper system which run under Android platform that used for your own wallpaper gallery application. With powerful and Responsive Admin Panel can manage unlimited category and image gallery, change user settings, etc. This application created by Android for client side and then PHP MySQL for Admin side. Run under Android platform which is the most popular operating system in the world. Using this application you can save your money and time in creating application for your own wallpaper application.

After you buy this item you will get project file of Android app template code, apk file, admin panel, sql database and also this documentation file to guide you installing this app.

Getting Started

Before doing installation, you need to do the following things :
  1. Have a web hosting to store admin panel files and SQL database. Because this documentation using cpanel, a web hosting with cpanel is recommended, if you want work with offline mode, you can use local webserver like xampp, ect with latest version
  2. Install tools that requires for developing Android app such as Java Development Kit and Android Studio as the Official Android IDE. For updated tutorial about how to install them, you can see and Download from Android Developer website.
  3. If you do all things above, you can go to the next step.

 

How to Import the Project in Android Studio?

  1. Open Android Studio > Import Project (Eclipse ADT, Gradle, ect.)
  2. New window will opened > Browse to directory where you place "MaterialWallpaper" project  > Ok
  3. Wait for few minutes until all process import has finished.

 

How to Rename the Package Name?

  1. First click once on your package and then click setting icon on Android Studio.
  2. Close/Unselect Compact Empty Middle Packages
  3. Then, right click your package and rename it.
  4. Right click on the root folder of your project.
  5. Click "Open Module Setting".
  6. Go to the Flavours tab.
  7. Change the application ID with same name of your package name before > Press OK.

 

    

How to Change Application Name?

  1. Open Android Studio > app > res > values > strings.xml
  2. Select 'strings.xml'
  3. Enter your app name inside "app_name" string tag :
  4. <string name="app_name">Material Wallpaper</string>

How to Change Header App Color?

  1. Open Android Studio > app > res > values > colors.xml
  2. Select 'colors.xml'
  3. Enter your color code inside "header" color tag :
  4. <color name="ColorPrimary">#448AFF</color>
  5. This will change the toolbar / header app & app main color.  

How to Configure Database and Webservice In Your Server?

Database creation

  1. Open phpmyadmin
  2. Create database 'wallpaper'
  3. Import database from donwload package in 'db' folder

Database Configuration

  1. After creating database configure the connection file, open 'variables.php' in folder 'includes' which is in the includes folder of your package.
  2. That’s all,now run the webservice : http://localhost/material_wallpaper or your domain name. 
  3. configure your database setting :
  4. $host ="localhost";
$user ="your_mysql_username";
$pass ="your_password";
$database = "your_database_name";

 

 

  1. If you upload your php code to root folder, as example www.yourdomain.com, please open 'connect_database.php' and 'close_database.php'  and change the code from : 
  2. include($_SERVER['DOCUMENT_ROOT'].'/material_wallpaper/includes/variables.php');
  3. Change with this :
  4. include($_SERVER['DOCUMENT_ROOT'].'/material_wallpaper/variables.php');
  5. But if you upload your php code into your directory, as example : www.yourdomain.com/androidwallaper/, just keep the code and don't change it.
  6. include($_SERVER['DOCUMENT_ROOT'].'/material_wallpaper/includes/variables.php');
  7. '/androidwallpaper' is indicated the name of your directory, if you change to another name, you should change the name in the code become your own directory name.

How to Change AdMob Publisher Id?

  1. Open Android Studio > app > res > values > strings.xml
  2. Select 'strings.xml'
  3. Enter your AdMob publisher id inside "admob_publisher_id" string tag for banner ads
  4. Enter your AdMob publisher id inside "admob_interstitial_id" string tag for interstitial ads
  5. <string name="admob_publisher_id">your admob  banner id here</string>
    <string name="admob_interstitial_id">your admob interstitial id here</string> 

 

How to change Push notification Application Id?

  1. First open parse.com & Register and Login with your Account.
  2. Press button Create New App in Right Side.
  3. Enter Your App Name
  4. Copy Application Id, Client key.
  5. Open Android Studio > app > res > values > strings.xml
  6. Enter your Parse Application Id inside "parse_application_id" string tag
  7. Enter your Parse Client Key Link inside "parse_client_key" string tag 

How to Change the Application Icon?

  1. Open Android Studio > app > res > drawable > ic_launcher.png
  2. Rename your icon with ic_logo_app.png and paste > replace in drawable-hdpi folder 
  3. Do the same with on /drawable-mdpi/, /drawable-xhdpi/, and /drawable-xxhdpi/ 

How to Change App About Information?

  1. Open Android Studio > app > res > values > strings.xml
  2. Select 'strings.xml'
  3. Enter your about information below in <!-- About Content --> string tag 
  4. For advanced change about content, you also can go to res > layout > about.xml

How to change More App Link?

  1. Open Android Studio > app > res >values > strings.xml
  2. Enter your Play Store Apps Link inside "play_more_apps" string tag:
    <string name="play_more_apps">Your App Url</string>

How to change Navigation Drawer Header Image and Icon?

  1. Open Android Studio > app > res > drawable 
  2. Find icon with name nav_drawer_header_image.png and nav_drawer_icon.png
  3. Replace with your image in drawable and drawable-hdpi folder 

Graphics

You will find all image into /res/drawable/, /res/drawable-mdpi/, /res/drawable-hdpi/, /res/drawable-xhdpi/ folder. Just create your own graphics and replace those files. Create seperate graphics of different device screen sizes & place them into other drawble folders (drawable-hdpi & drawable-xhdpi) with same file name found at drawable-mdpi folder.
